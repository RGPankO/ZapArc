Lightning Tipping Settings

Wallet Configuration

Recommended for most users. Provides full wallet functionality with automatic LNURL generation.
For advanced users who want to use their own Lightning wallet or service.
Enter your LNURL-pay address to receive tips directly to your external wallet. Must start with "lnurl" and contain valid bech32 characters.

Default Amounts

Posting Amounts (sats)

Default amounts included in your tip requests when posting content. These will be suggested to people who want to tip you.
Tip: Use ascending amounts (e.g., 100, 500, 1000) for better user experience

Tipping Amounts (sats)

Your preferred amounts when tipping others. These appear alongside the content creator's suggested amounts.
Maximum: 100,000,000 sats (1 BTC). All amounts must be unique.

Facebook Group Management

Posting Mode

Control how tip requests are added to Facebook posts and comments.
Enable tip request appending on all Facebook groups and pages.
Only enable tip request appending in specific Facebook groups that you choose.

Allowed Facebook Groups

Manage which Facebook groups allow tip request appending. Groups are automatically detected when you visit them, or you can add them manually.
You can enter a group ID (numbers only) or paste a Facebook group URL like facebook.com/groups/123456789

Interface Settings

Shows a floating menu on websites for quick access to wallet functions and domain management.
Automatically locks your wallet after inactivity for security. "Never" is not recommended for security reasons.
